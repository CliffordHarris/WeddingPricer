# WeddingPricer
WeddingPricer is in the process of being reactified for learning purposes.


*Previous static .html, .js, .css files have been moved to 'static' folder.*


#### To Run the App
- `git clone https://github.com/CliffordHarris/WeddingPricer.git` and `cd` to the directory you cloned it to.

- `yarn install`

- `yarn start`

***Note: Will require NPM, Yarn, Node*
